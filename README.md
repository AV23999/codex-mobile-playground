# Codex Mobile Playground

This repository is a **fresh sandbox** for letting Codex / Claude / Ruflo generate mobile apps from scratch.

## How to use this repo

- Start a new conversation in your AI tool (Codex, Claude Code, etc.).
- Give it this repo URL:
  - `https://github.com/AV23999/codex-mobile-playground`
- Tell it explicitly:
  - "Initialize a new React Native (Expo) mobile app in this repo from scratch. Create all folders and files, commit them, and keep using this repo for future updates."

The repo is intentionally empty (besides this README) so the AI can decide the full structure.

You can later add backend or shared-library repos if needed.
